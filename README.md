# Iris Report v9 — Local Proxy

## Стартиране
1) Инсталирай Node.js (LTS).
2) В тази папка: копирай env.example -> .env и попълни ключовете.
3) Стартирай: node server.mjs
4) Отвори: http://localhost:5173/

## Ако си отварял index.html с двойно кликване
Вече има "Директно към API" режим:
- отвори index.html директно от файла;
- избери "Директно към API (браузър)" и въведи OpenAI/Gemini ключовете в полетата под падащото меню;
- сложи отметка "Запомни режима и ключовете", ако искаш да се зареждат автоматично при следващо отваряне;
- работи без server.mjs, но има предвид че ключовете се изпращат от самия браузър.
